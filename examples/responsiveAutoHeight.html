Responsive Auto-Height

When the target "responsiveWidth" (or less) is reached, ScrollViews enters "Responsive Mode." This will add the class, "svs-responsive" to the document's body element. Finally, because the class "svs-auto-height-responsive" is on this section DIV, it will become taller than the viewport using height: auto.


The responsiveWidth is set to 900px in this demo.
